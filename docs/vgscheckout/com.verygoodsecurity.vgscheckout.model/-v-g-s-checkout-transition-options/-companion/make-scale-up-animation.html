@JvmStatic
fun makeScaleUpAnimation(source: View, startX: Int, startY: Int, width: Int, height: Int): VGSCheckoutTransitionOptions
Content copied to clipboard

Create an VGSCheckoutTransitionOptions specifying an animation where the checkout is scaled from a small originating area of the screen to its final full representation.

Return

VGSCheckoutTransitionOptions object that can be used as a parameter of com.verygoodsecurity.vgscheckout.VGSCheckout.present function.

Parameters

source

the View that the new activity is animating from. This defines the coordinate space for startX and startY.

startX

the x starting location of the new activity, relative to source.

startY

the y starting location of the activity, relative to source.

width

the initial width of the new activity.

height

the initial height of the new activity.
