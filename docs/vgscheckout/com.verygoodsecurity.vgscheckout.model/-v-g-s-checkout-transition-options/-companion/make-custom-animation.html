@JvmStatic
fun makeCustomAnimation(context: Context, enterAnimId: Int, exitAnimId: Int): VGSCheckoutTransitionOptions
Content copied to clipboard

Create an VGSCheckoutTransitionOptions specifying a custom animation to run when the checkout is displayed.

Return

VGSCheckoutTransitionOptions object that can be used as a parameter of com.verygoodsecurity.vgscheckout.VGSCheckout.present function.

Parameters

context

who is defining this. This is the application that the animation resources will be loaded from.

enterAnimId

a resource ID of the animation resource to use for the incoming activity. Use 0 for no animation.

exitAnimId

a resource ID of the animation resource to use for the outgoing activity. Use 0 for no animation.
